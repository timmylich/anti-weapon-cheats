## [Read in English](README.md)

## Что это?

Данный инклуд исправляет ошибки, связанные с использованием оружия и препятствует использованию некоторых читов в SA-MP серверах.

### Возможности:

1. **Проверка скорости выстрелов** - предотвращает использование читов типа +C, Rapid и других ускорителей стрельбы
2. **Проверка скорости ударов в ближнем бою** - препятствует использованию дамагеров для холодного оружия

## Подключение

Для работы достаточно просто подключить файл `anti-weapon-cheats.inc` в свой код:

```cpp
#include <anti-weapon-cheats>
```

### Настройка перед подключением

Вы можете отключить отдельные проверки, определив соответствующие константы перед подключением инклуда:

```cpp
// Отключить проверку скорости выстрелов
#define AWCheats_FireRateOver 0

// Отключить проверку скорости ударов в ближнем бою
#define AWCheats_MeleeRateOver 0

#include <anti-weapon-cheats>
```

## Как это работает

### Проверка скорости выстрелов
Скрипт отслеживает время между выстрелами для каждого игрока и каждого типа оружия. Если игрок стреляет слишком быстро (быстрее установленного лимита), выстрел блокируется.

### Проверка скорости ударов
Аналогично стрельбе, скрипт контролирует скорость ударов холодным оружием и блокирует слишком частые атаки.

## Совместимость

Скрипт использует стандартные SA-MP колбэки:
- `OnPlayerWeaponShot` - для проверки выстрелов
- `OnPlayerGiveDamage` - для проверки ударов

Все колбеки правильно перехватываются.

## Настройка лимитов

Если вам нужно изменить лимиты для определенного оружия, отредактируйте массив `s_MaxWeaponShootRate` в файле инклуда:

```cpp
static s_MaxWeaponShootRate[] = {
    250, // 0 - Fist
    250, // 1 - Brass knuckles
    // ... измените нужные значения
};
```
